Bazzite

Bazzite is a gaming-focused, immutable Linux distribution based on Fedora and designed to mimic the Steam Deck experience. It prioritizes a stable and secure environment by using Universal Blue, an immutable operating system framework. Bazzite is designed for both desktop and handheld devices, including the Steam Deck, and aims to provide a seamless gaming experience out-of-the-box.

Nobara

Nobara Linux is a modified version of Fedora designed to be more user-friendly, especially for gaming, streaming, and content creation. It includes features and software that are often missing or require extra setup in a standard Fedora installation, such as Wine dependencies, OBS Studio, third-party codecs, and Nvidia drivers. Nobara aims to provide a more "point-and-click" experience, simplifying tasks for users who prefer graphical interfaces and reducing the need to delve into the terminal for common tasks.

PikaOS

PikaOS is a gaming focused distro built on a Debian base. It aims to provide a seamless out-of-the-box gaming experience with performance enhancements and a user-friendly setup. PikaOS features a custom-tweaked kernel, optimized packages, and up-to-date drivers. It also includes a suite of custom tools for system management and a choice of desktop environments (GNOME, KDE Plasma, Hyprland).
